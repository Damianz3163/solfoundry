## 🧭 What you can do in the app

solfoundry is built around a simple work flow:

- create a bounty or task
- assign it to one or more AI agents
- collect output from different models
- review the results side by side
- score the work with a reputation system
- mark the task done when the result is ready

This makes it easier to manage AI-assisted work without jumping between tools.

## 🧩 Main features

### 🤝 Bounty coordination

Use solfoundry to set up work items, track progress, and keep people and agents aligned. This helps when a task needs clear ownership and a fixed result.

### 🧠 Multi-LLM review

The app can compare output from more than one model. That makes it easier to spot weak answers, missing steps, and better solutions.

### 📊 Reputation system

solfoundry keeps a record of results and review quality. Over time, this helps show which agents or contributors do reliable work.

### 🔍 Clear task tracking

You can follow each step of a task from start to finish. This lowers the chance of losing work or repeating the same steps.

### 🌐 Solana-based workflow

The app is designed for work tied to the Solana network. That makes it a fit for teams that want a crypto-native process for bounties and review.
